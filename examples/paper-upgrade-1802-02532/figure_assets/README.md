# Figure Assets

This folder contains the dedicated high-resolution assets for the figures used in the upgraded paper.

## Included Figures

- `serialization_view/serialization_view.pdf`
  Cleanly named vector PDF suitable for print workflows
- `serialization_view/serialization_view_standalone.pdf`
  Build-preserved PDF emitted directly from the standalone source
- `serialization_view/serialization_view.svg`
  Vector SVG for web or document reuse
- `serialization_view/serialization_view-1.png`
  600 DPI raster export
- `serialization_view/serialization_view_standalone.tex`
  Standalone LaTeX source used to regenerate the asset
- `probe_curves/probe_curves.pdf`
  Cleanly named vector PDF for the computed locality and context-budget curves
- `probe_curves/probe_curves_standalone.pdf`
  Build-preserved PDF emitted directly from the standalone source
- `probe_curves/probe_curves.svg`
  Vector SVG for web or document reuse
- `probe_curves/probe_curves-1.png`
  600 DPI raster export
- `probe_curves/probe_curves_standalone.tex`
  Standalone LaTeX source used to regenerate the computed figure

The paper itself contains LaTeX-side versions of these figures in [../paper/figures/FIG_serialization_view.tex](../paper/figures/FIG_serialization_view.tex) and [../paper/figures/FIG_probe_curves.tex](../paper/figures/FIG_probe_curves.tex).
