# Figure Assets

This folder contains high-resolution standalone exports of the paper's figures.

## Included Assets

- `timeline/`
  Event timeline of the Wilkinson-Jefferson-Frederick corpus.

- `network/`
  Weighted connection network showing Frederick's bridge role.

Each figure folder is expected to contain:

- standalone `.tex`
- vector `.pdf`
- vector `.svg`
- `600` DPI `.png`
