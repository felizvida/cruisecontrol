# Data Package

This example ships the sample data needed to rebuild the paper's results tables.

## Sample Data

- Local synthetic benchmark: [../results/demo_benchmark.json](../results/demo_benchmark.json)
- Machine-readable manifest: [dataset_manifest.json](dataset_manifest.json)

## What This Data Represents

The benchmark JSON is a synthetic demonstration artifact. It defines:

- evaluation regimes
- per-method mission return
- distance traveled
- cost of transport
- falls per 100 meters
- adaptation-compute energy
- cost-sweep and ablation slices

The paper is explicit that these values are illustrative and exist to demonstrate the end-to-end workflow rather than to claim new robotics evidence.
