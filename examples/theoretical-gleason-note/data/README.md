# Data Package

This theoretical example has no external experimental dataset.

The relevant inputs are:

1. the reference papers listed in `source_manifest.json`
2. the generated numeric sanity-check outputs written by the local scripts

## Included Files

- [source_manifest.json](source_manifest.json)
  Machine-readable manifest of the theoretical sources and generated artifacts

- [../results/reconstruction_checks.json](../results/reconstruction_checks.json)
  Generated numeric sanity checks

- [../results/check_summary.csv](../results/check_summary.csv)
  Tabular summary of the generated checks

## Interpretation

The generated files are **not** evidence for the theorem in place of the proof. They are reproducible illustrations that the recovered representing matrix behaves exactly as the theorem predicts on random sampled examples.
