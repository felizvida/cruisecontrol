# Round 08 Review

Reviewed artifact: `paper/main_round8.pdf`

Score: `8.6 / 10`

Verdict: `Accept with Minor Revision`

Main criticisms:

1. The paper is admirably honest about relying on a curated marker panel, but that limitation is still described qualitatively rather than quantitatively.
2. A reviewer should not have to infer how small the curated panel is relative to the full region-count mass.
3. Quantifying that sparsity would actually strengthen the paper, because it would show the authors are not overselling a narrow evidentiary slice.

Required fixes for Round 09:

1. Compute the curated marker-panel fraction of the total regional count mass.
2. State that quantity in the abstract or discussion so the scope limitation is numeric.
3. Use the number to sharpen the distinction between a broad atlas claim and the narrower legibility claim this note actually makes.
