\section{Reproducibility Appendix}

\subsection{Local files}

The paper package is backed by:
\begin{itemize}
  \item a source manifest in \texttt{data/source\_manifest.json},
  \item a curated evidence file in \texttt{data/curated\_evidence.json},
  \item a Python analysis script in \texttt{results/analyze\_region\_proteome.py},
  \item an R figure-generation script in \texttt{results/render\_figures.R},
  \item generated summary files in \texttt{results/},
  \item a family-level robustness file in \texttt{results/group\_consistency.csv},
  \item build scripts in \texttt{code/}.
\end{itemize}

\subsection{Formulas}

The main derived formulas are the Jaccard overlap, specialization fractions, marker-level stabilized log-bias, Wilson intervals for axis-level alignment fractions, and an exact test on the $2\times2$ matched-versus-spillover table.

\subsection{Scope note}

The computation uses values recoverable from the published article and repository metadata. It does not claim a full raw-data reprocessing pipeline.

\subsection{Exact rebuild path}

The package can be rebuilt locally with:
\begin{verbatim}
cd examples/codex-zebrafish-brain-paper-14rounds
./code/build_inputs.sh
./code/build_paper.sh
\end{verbatim}

The analysis step should regenerate, at minimum:
\begin{itemize}
  \item \texttt{results/summary\_metrics.json},
  \item \texttt{results/axis\_summary.csv},
  \item \texttt{results/group\_consistency.csv},
  \item the figure files under \texttt{figures/}.
\end{itemize}

The paper step should regenerate \texttt{paper/main.pdf}.

\subsection{Marker-panel curation rule}

The marker panel is intentionally conservative. A family enters the panel only if the source proteomics paper names it in the biological interpretation of telencephalon or optic tectum and if that interpretation can be connected to established zebrafish regional function in the cited background literature. The curated counts and family labels are materialized in \texttt{data/curated\_evidence.json}, while the corresponding per-family summaries appear in \texttt{results/marker\_bias.csv} and \texttt{results/group\_consistency.csv}.
