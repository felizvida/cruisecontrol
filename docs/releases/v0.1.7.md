# v0.1.7

This release raises the paper-writing bar in two ways: it makes the repo's prose standard explicit, and it folds that standard into the default writing workflow.

## Highlights

- Added `classic-biology-prose`, a new paper-writing skill that pushes prose toward the clarity of Monod and Crick, the narrative movement of Jacob, the personality of Brenner, and the reflective cadence of Stent.
- Wired that style into the default paper-writing route, so both Codex and OpenCode workflows inherit the same human-writing standard unless explicitly overridden.
- Updated the GitHub-facing docs to explain the new writing default in the README, Quick Start, route guide, and narrative walkthrough.
- Improved privacy in the `paperreview.ai` client by redacting submission emails from saved metadata and keeping only the domain.
- Refreshed the Codex zebrafish paper package through `main_round26.pdf`, including later prose-focused revisions and an explicit record of the blocked `paperreview.ai` round-26 submission attempt.
