# v0.1.3

Native OpenCode runtime reliability release.

## Included

- Switched the default `codex` MCP entry to a repo-local wrapper script
- Isolated OpenCode's Codex reviewer path from global Codex MCP config and stale Codex state
- Added repo-local Codex profiles for the copied research workflows
- Increased OpenCode MCP timeout so longer native literature and review calls can finish
- Updated README and quickstart docs to describe the new local runtime setup

## Validation

- `opencode mcp list` shows `codex` connected through the wrapper
- Native `opencode run --command research-lit ...` completed successfully on this machine
- A native smoke-test artifact was written under `.local_artifacts/native-opencode-smoke/`

## Notes

- This release is still optimized for the current machine first
- The runtime-isolation wrapper is local-repo infrastructure, not upstream ARIS behavior
- Optional `zotero` and `obsidian-vault` integrations remain disabled by default
