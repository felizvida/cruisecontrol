# v0.1.0

Initial public release of the OpenCode port of ARIS.

## Included

- OpenCode-native command wrappers for the ARIS workflows
- Ported research skills under `.opencode/skills/`
- Local `AGENTS.md` rules for MCP naming and artifact handling
- `opencode.jsonc` configured for local Codex MCP on this machine
- Quickstart onboarding guide
- Narrative auto-research guide
- Upstream attribution and provenance docs

## Notes

- This release is optimized for the current machine and pins Codex to `/opt/homebrew/bin/codex`.
- Intermediate artifacts are kept local and are not intended to be committed.
- Upstream source remains credited in `README.md`, `NOTICE.md`, and `UPSTREAM.md`.
